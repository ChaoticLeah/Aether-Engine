Getting Components:

You can get components of an object with the getComponent(str) method. This will return an array of components relevant to the type you asked for. parent.getComponent("Script") would return an array of all the scripts on the parent object. If you want to get all the components you can just use parent.components, this is an array of all the object's components. An example of this can be seen below:

script.js

A list of all the names of objects:

parent.getComponent("Rectangle Component") - This returns all the Rectangle Components.
parent.getComponent("Script") - This returns all the Script Components.
parent.getComponent("Textured Component") - This returns all the Texture Components.
